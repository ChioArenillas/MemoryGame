# Meme Generator

## 🧩 Description

Memory Game is an accessible React-based application where users can customize the game by selecting the number and type of cards. Emojis are dynamically fetched from an API to generate the game board, creating a unique experience each time.

The project places a strong emphasis on accessibility, ensuring that all users can interact with the game through keyboard navigation and screen reader support using ARIA attributes.

## 🚀 Live Demo

https://memory-game-two-eta-84.vercel.app/

## 🚀 Technologies

* React
* JavaScript (ES6+)
* HTML5
* CSS3
* Vercel (deployment)

## 🎯 Features

* Select number and type of cards
* Dynamic emoji fetching from an API
* Randomized board using Fisher-Yates shuffle
* Memory matching game logic
* Accessible interactions (ARIA, focus management, keyboard navigation)
* Responsive and interactive UI

## 🧠 Key Concepts

* React state management (useState, useEffect)
* Accessibility (ARIA attributes, aria-live, focus control)
* Array manipulation and algorithms (Fisher-Yates shuffle)
* Event handling
* Dynamic rendering
* Immutable state updates
